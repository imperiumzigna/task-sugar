# Usage
<!-- usage -->
```sh-session
$ npm install -g task-sugar
$ tsr COMMAND
running command...
$ tsr (-v|--version|version)
task-sugar/0.0.0 darwin-x64 node-v12.18.2
$ tsr --help [COMMAND]
USAGE
  $ tsr COMMAND
...
```
